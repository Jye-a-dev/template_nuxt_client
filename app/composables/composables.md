# composables

EN: Stores Nuxt/Vue composables for reusable stateful logic. Use this folder for functions like authentication state, filters, and shared reactive helpers.

VI: Chứa các composable Nuxt/Vue cho logic có trạng thái dùng lại. Dùng cho các hàm như trạng thái đăng nhập, bộ lọc và helper reactive dùng chung.
