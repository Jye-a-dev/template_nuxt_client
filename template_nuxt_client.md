# template_nuxt_client

EN: Root workspace for the Nuxt client template. It contains project configuration, package scripts, documentation, public assets, application source code, and server-side Nuxt utilities.

VI: Thư mục gốc của template Nuxt client. Nơi chứa cấu hình dự án, script package, tài liệu, tài nguyên public, mã nguồn ứng dụng và các tiện ích phía server của Nuxt.
